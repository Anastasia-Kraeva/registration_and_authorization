Цель задания сохранить токен в глобальном стейт менеджере (Redux).

Все необходимые АПИ находятся в разделе User - по ссылке http://188.166.119.86:8080/swagger/

Необходимо реализовать стандартную регистрацию юзера с представленным ниже полями( http://188.166.119.86:8080/swagger/ раздел User).
Когда пользователь заполнил все необходимые поля, отправить данные на сервер, после чего, перенаправить юзера на страницу авторизации. (потому что, токен дается только при авторизации)

Внимание! Для подтверждения аккаунта нужно передать специальный ключ, который вы получаете в урле браузера когда переходите по ссылке с почты. (register-verify?key={email_confirmation.key})
Нужно сделать автоматический запрос на апи user/register/verify и отправить полученный ключ на сервер для активации аккаунта. После чего вы сможете перейти к части авторизации пользователя в приложение.

Реализовать авторизацию ( http://188.166.119.86:8080/swagger/, подраздел User -   PreLogin / Login  ).
Когда пользователь заполнил все необходимые поля, данные нужно отправить на сервер, после чего, перенаправить юзера на главную страницу.

Авторизация состоит из 2-х этапов.
Первый этап это отправка данных на север после чего отправится сообщение на почту пользователя.

Второй этап подтверждающий код. Нужно реализовать форму подтверждения кода.
Когда запрос будет выполнен успешно, апи вернет токен (refresh and access)

Реализовать выход из учетной записи пользователя ( http://188.166.119.86:8080/swagger/  , раздел User - Logout  ).
